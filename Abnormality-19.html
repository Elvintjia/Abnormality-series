Abnormality-19

Alias: Cancel calendar

Danger level: 1

Table of contents

  1. Description
  2. Test logs
  3. Incidents
  4. Addenda
  5. Origin

Description

Abnormality-19 is a small blue calendar. Abnormality-19 changes itself depending on the current year, month, and day. Anything events written in Abnormality-19 will never happen. Something will always happen that either causes people to be unable to attend the event, or something causes the event to be impossible to be continued. Anything written in Abnormality-19 will disappear after said incident happens. Abnormality-19 follows the Gregorian calendar.

Test logs

Test 19-1

Experiment: An ongoing event was written on Abnormality-19.

Result: The event didn't get cancelled.

Test 19-3

Experiment: A predicted natural disaster was written on Abnormality-19.

Result: Said disaster never happened.

Test 19-4

Experiment: An event without a specific date was written.

Result: The event was cancelled.

Test 19-5

Experiment: A sticky note with an event written on it was pasted on the calendar.

Result: The sticky note disappeared. The event written was cancelled.

Test 19-6

Experiment: Random doodles were scribbled on Abnormality-19.

Result: Nothing happened. The doodles didn't disappear until erased by a staff member.

Test 19-7

Experiment: Non-event things were written on Abnormality-19.

Result: The written things stayed on Abnormality-19 until a staff member erased them.

Test 19-8

Experiment: An event was written on Abnormality-7. The same event was written on Abnormality-19.

Result: The event happened.

Incidents

Incident 19-1

Test 19-2 has been deleted from this report. A big event had been written on Abnormality-19. A disaster happened which caused death and injuries. Test 19-2 is only available to high-level personnel. Testing on Abnormality-19 now must follow a strict protocol and requirements to prevent similar incidents. Big events are now forbidden from being written in Abnormality-19, except to prevent disasters.

Addenda

Addendum 19-1

Through a series of tests, several new things about Abnormality-19 was discovered. Abnormality-19 is unable to cancel an ongoing event. An event can only be cancelled before it starts. Abnormality-19 can cancel any event no matter how big, as long as said event was predicted and written on Abnormality-19. Natural disasters and events without specific dates can also be prevented with Abnormality-19. An event written on a sticky note pasted on Abnormality-19 will be cancelled. Said sticky notes will disappear like most writings on Abnormality-19. Non-event text written on Abnormality-19 will stay on Abnormality-19 will stay on it permanently, unless erased.

Origin

Abnormality-19 was found in the aftermath of a shootout with the cult. Watchmen found an operation the cult was doing. Commandos were sent to intercept them. Civilian casualties were minimized by preemptive evacuation. The local government had given permission for the organization to act. A shootout ensued. Due to being overwhelmed, cultist escaped, leaving their weapons and tools behind. Abnormality-19 was found among the the things the cultist left behind.
